# Climate Risk Heatmap

A browser-based scenario model that visualizes climate risk intensity across a synthetic region grid.

## Highlights
- Interactive controls for warming, rainfall volatility, and adaptation budget.
- Procedural risk engine balancing hazard and resilience factors.
- Heatmap rendering with live risk summary metrics and alert levels.

## Stack
- HTML5 Canvas
- Vanilla JavaScript
- Responsive CSS interface

## Run
Open `index.html` in your browser.
